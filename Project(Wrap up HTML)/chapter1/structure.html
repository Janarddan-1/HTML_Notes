Basic Structure:

  1. <html>...</html> : html tag specify from where to where is the actual html content. It the the parent tag of <head> & <body> tag.
  2. <head>...</head> : head tag is the 1st child of <html> tag. It contains the information(primarily holds information for machine processing, not human-readability) obout the page, meta deta, style, scripts.
  3. <body>...</body> : body tag is the 2nd child of <html> tag. It contains the main content of the webpage.
  4. "! + Enter Key " : We can create the basic structure for an html document by this shortcut.(this auto generated lines are made by emmet)
  5. Emmet : Emmet is a plugin for many popular text editors which greatly improves HTML & CSS workflow (comes built in inside VS-Code). Emmet official cheat-sheet.
  6. "alt + shift + down arrow key" - to make duplicate line of the line where the cursur was
  7. "ctrl + space" - for attribute suggestion or value of attribute suggestion of any HTML element.

Example: 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Html Document Structure</title>
        </head>
        <body>
            // Main Content

        </body>
        </html>

Html head elements:

  1. <!DOCTYPE html> : This line represent that the document type is HTML.
  2. in <html lang="en">, the .. lang="en" .. represent that the HTML document is in English language
  3. <title>...</title> : The title tag give the web page's tab a name.
  4. <style>...</style> : style tag is used to add internal css to the html document
  5. <script>...</script> : script tag is used to add javacript in the html document
  6. link : The link tag specifies relationships between the current document and an external resource. This element is most commonly used to link to stylesheets, but is also used in other linking(see the Attribute section in the MDN docs). Eg: (type "link:css" + Enter key) <link rel="stylesheet" href="style.css">
  7. meta: meta tags are HTML tags used to provide additional information about a page to search engines and other clients. Clients process the meta tags and ignore those they don't support. Most of the meta tag contains 2 attribute "name" & "content".
  8. <meta charset="UTF-8">: Specifies the character encoding for the HTML document. UTF-16 support more languages and emoji than UTF-8
  9. <meta name="viewport" content="width=device-width, initial-scale=1.0"> : meta viewport gives the browser instructions on how to control the page's dimensions and scaling.
  10. <meta http-equiv="refreash" content="30"> : this meta tag makes this webpage refreash in every 30 sec (Normally not used) (Preferred way: It can be alternatively done by javascript)
  11. <meta name="keywords" content="HTML, web"> : Used for SEO
  12. <meta name="description" content="This is a HTML head's tag demo"> : Used for SEO

class & id

  1. class and id both works almost the same and used to target or select HTML element.
  2. For accessing HTML element with Class we have to use a dot(.) symbol before the class name. Eg: " .id_name ".
  3. For accessing HTML element with ID we have to use a hash(#) symbol before the id name. Eg: " #id_name "
  4. According to html convention class name can be used by multiple HTML elements, while an id name must only be used by one HTML element within a page. But following this convention is not necessary
